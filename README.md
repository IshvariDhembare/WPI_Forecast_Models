# WPI_Forecast_Models
This repository focuses on forecasting price volatility in agricultural commodities, specifically the Wholesale Price Index (WPI) of potatoes. Using ARIMA, SARIMA, Holt-Winters, ARMA, and SVR models, we analyze price index data to identify the most accurate forecasting approach.
